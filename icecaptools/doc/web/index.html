Welcome to the home of the HVM

  • The HVM is an open source Java execution environment for resource constrained embedded platforms
  • It comes as an Eclipse plugin that translates Java source code into portable C code
  • The HVM supports SCJ Level 0, 1 and 2
  • The HVM reference manual is available here
