Project suggestions

The following is a list of options for development projects based on the HVM. Some of these projects have a size and complexity that makes them suitable for a semester project in an IT engineering course, some of the projects could even form the basis for a master thesis:
  • Garbage collection. Implement a reference counting garbage collector for the HVM
  • Wireless protocol. Using the CR16C, program the DIP from Java to implement a simple wireless data transmission protocol
  • MicroC/OSII. Create a native interface to the MicroC/OSII real time OS in order to implement the Thread and Semaphore concepts based on MicroC/OSII
